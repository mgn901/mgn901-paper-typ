# レポート・論文組版向けTypstテンプレート「mgn901-paper-typ」

「mgn901-paper-typ」は，レポート・論文組版向けのTypstテンプレートです．和文のレポートや論文の組版に用いられることを念頭に置いています．

使用方法の簡単な説明は[main.typ](main.typ)（と、そのPDFである[main.pdf](main.pdf)）に記述されています。
